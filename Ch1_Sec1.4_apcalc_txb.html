1. Special Trigonometric Limits

To solve many trigonometric limits, you need to know these two fundamental rules.

 

Rule 1

This rule is used for sine functions approaching a value of zero.

\[ \lim_{x \to 0} \frac{\sin x}{x} = 1 \]

This limit forms the basis for many other problems. It is crucial to remember that the argument inside the sine function must match the expression in the denominator.

Rule 2

This rule is used for cosine functions approaching a value of zero.

\[ \lim_{x \to 0} \frac{1 - \cos x}{x} = 0 \]

Similar to the first rule, the argument of the cosine function must match the expression in the denominator.

2. Using the Special Limits

When solving more complex problems, you often need to manipulate the expression to fit one of the special rules. This usually involves multiplying the numerator and denominator by a constant to match the terms.

Example A: Matching the Denominator

Find the value of the limit:

\[ \lim_{x \to 0} \frac{\sin(2x)}{x} \]

We can't use Rule 1 directly because the argument of the sine function is \(2x\), not \(x\). To fix this, we multiply the numerator and the denominator by 2.

\[ \lim_{x \to 0} \frac{\sin(2x)}{x} \cdot \frac{2}{2} = \lim_{x \to 0} 2 \cdot \frac{\sin(2x)}{2x} \]

By the special limit rule, we know that as a variable approaches zero, the limit of the sine of that variable over itself is 1.

\[ 2 \cdot \lim_{x \to 0} \frac{\sin(2x)}{2x} = 2 \cdot 1 = 2 \]

Example B: Applying Rule 2

Find the value of the limit:

\[ \lim_{x \to 0} \frac{1 - \cos(2x)}{x^2} \]

This limit resembles Rule 2. The correct approach for this problem is to use the double-angle identity:

\[ 1 - \cos(2x) = 2\sin^2(x) \]

Substituting this identity into the limit expression gives:

\[ \lim_{x \to 0} \frac{2\sin^2(x)}{x^2} = \lim_{x \to 0} 2 \left( \frac{\sin x}{x} \right)^2 \]

Using Rule 1, this simplifies to:

\[ 2 \cdot (1)^2 = 2 \]

3. Limits That Approach a Value Other Than Zero

Sometimes the variable approaches an integer instead of zero. In this case, we use a change of variable to transform the limit into a more familiar form.

Example C: Change of Variable

Find the value of the limit:

\[ \lim_{x \to 1} \frac{\sin(\pi(x-1))}{x-1} \]

Let \( u = x-1 \). As \(x \to 1\), \(u \to 0\). We can rewrite the limit in terms of \(u\).

\[ \lim_{u \to 0} \frac{\sin(\pi u)}{u} \]

Now we can use the same technique as in Example A by multiplying the numerator and denominator by \(\pi\).

\[ \lim_{u \to 0} \frac{\sin(\pi u)}{u} \cdot \frac{\pi}{\pi} = \lim_{u \to 0} \pi \cdot \frac{\sin(\pi u)}{\pi u} = \pi \cdot 1 = \pi \]

Check Yourself

Find \(\lim_{x \to 0} \frac{\tan x}{x}\).

Rewrite \(\tan x\) as \(\frac{\sin x}{\cos x}\):

\[ \lim_{x \to 0} \frac{\sin x / \cos x}{x} = \lim_{x \to 0} \frac{\sin x}{x \cos x} \]

Using the properties of limits, this becomes:

\[ \lim_{x \to 0} \frac{\sin x}{x} \cdot \lim_{x \to 0} \frac{1}{\cos x} = 1 \cdot \frac{1}{\cos(0)} = 1 \cdot \frac{1}{1} = 1 \]

Find \(\lim_{x \to 0} \frac{1 - \cos(3x)}{x^2}\).

Use the half-angle identity \(1 - \cos(2\theta) = 2\sin^2(\theta)\). Here, \(2\theta = 3x\), so \(\theta = \frac{3x}{2}\).

\[ \lim_{x \to 0} \frac{2\sin^2(\frac{3x}{2})}{x^2} = 2 \lim_{x \to 0} \left( \frac{\sin(\frac{3x}{2})}{x} \right)^2 \]

Multiply inside the parenthesis to match the denominator:

\[ 2 \lim_{x \to 0} \left( \frac{\sin(\frac{3x}{2})}{x} \cdot \frac{\frac{3}{2}}{\frac{3}{2}} \right)^2 = 2 \lim_{x \to 0} \left( \frac{3}{2} \cdot \frac{\sin(\frac{3x}{2})}{\frac{3x}{2}} \right)^2 \]

Using Rule 1, this simplifies to:

\[ 2 \left( \frac{3}{2} \cdot 1 \right)^2 = 2 \cdot \frac{9}{4} = \frac{9}{2} \]
